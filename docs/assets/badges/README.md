# README badge SVGs

Repo-hosted images for the README hero row (Scorecard, License, Release). See [REPO_SETUP.md §10](../../../REPO_SETUP.md#readme-badge-images-repo-hosted) for refresh commands.

Do not point these image `src`s at `img.shields.io` or `api.scorecard.dev` — those CDNs intermittently fail (timeout / 5xx) and break the storefront.
